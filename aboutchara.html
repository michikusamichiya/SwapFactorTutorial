日本語/Japanese

文字式とは

文字式とは、数字と文字を組み合わせてあらわされる式のことです。

...とは言われても、「文字ってなんだ」という話です。
例えば、こんな問題を解いてみましょう。
小学生の時にもこんな問題を解きませんでしたか?
Aさんはみかんを10個持っています。Bさんはりんごを\(\square\)個
持っています。みかんとりんごの個数の合計はいくつですか?
この四角には、どんな数も入りますよね?
1個、2個、10個、100個、もしくは0個。どんな数を入れることもできます。
(※正確に言うと\(\square\in\mathbb{N}\), 四角に入るべき数は自然数である必要がある)
このように、「分からない数」を、四角のような文字としておく事があります。
あ、ちなみに答えは\(10+\square\)です。

...というのは小学生までの話。

中学生からの「文字式」とは

今までは四角や三角などの文字を使ってきましたが、
中学生になったら、何を使うでしょう?

答えは「アルファベット」。

ただ、アルファベットを使うときにはちょっとした決まりがあります。
この問題の例を基に考えてみましょう。
Aさんはみかんを\(a\)個、持っています。また、BさんはAさんの3倍の個数のりんごと\(b\)個のみかんを持っています。
Aさんのみかんと、Bさんのみかんとりんごの合計はいくつですか?
まず、この問題を解くために、Bさんがりんごを何個、みかんを何個持っているかを考えましょう。
  • まず、Aさんはりんごを\(a\)個持っている。
  • Bさんは「Aさんのりんごの3倍の個数」のりんごを持っている。
  • また、Bさんは\(b\)個のミカンを持っている。
これらを踏まえると、答えは
\[a+3\times a + b\] 個ということになります。
実はこのままだと正解ではありません。。

文字式のルール

では、その「ルール」とは何でしょうか。これです。
  • 掛け算の記号\(\times\)は省略します。
    掛け算の記号が残っていると計算途中とみなされるので、減点されます。
  • 数字と文字の並びは数字が先。
    \(a\times 4\times b=4ab\)のように、項ごとに数字を一番前に持ってくるようにしましょう。
    \(a4b\)のように、数字を記号の間や途中においてはいけません。
  • 文字同士の積の並びはアルファベット順。
    これはあまり重要ではありませんが、
    \(x\times a\times z\times b\)のような掛け算は
    アルファベット順に並べましょう。答えは\(abxz\)となります。
    \(xazb\)としてもあまり影響はありませんが...。
  • 同じ文字の積(掛け算)は指数で表す。 例えば、\(a\times a\times b\times c\times c\times c=a^2bc^3\)。
    \(a\times a=aa\)とするのではなく、文字の右上に同じ文字が何文字あるかというのを
    書きます。ただし数字の場合は指数を回答に載せないようにしましょう。(あまり大きな数でない限り...)
  • 割り算は分数を使う。 たとえば、\(2\times a\div b \times c=\dfrac{2ac}{b}\)のように、割り算の記号を使わないようにしましょう。
    分数を使いましょう。
  • 同類項はまとめる。
    同じ文字が同じだけ含まれる(たとえば、\(2ab+3ab\)、\(3a^3b^2c-4a^3b^2c\))項を、まとめておきましょう。
    たとえば、\(2ab+3ab=5ab\)、\(-a^3b^2c\)
  • 小学校のころ使っていた帯分数は使わないようにしましょう。基本は仮分数で。
  • (定数をあらわす文字は数字の次、未知数は文字の前におきましょう。このゲームでは出てきませんが、
    円周率などの定数\(\pi\)は数字の後に書きましょう。)
これらの法則に従って、 \[a+3\times a + b\] をちゃんとした式に直しましょう。
まず、掛け算の記号を消すと、こうなります。
\[a+3a+b\]
同類項をまとめると、こうなります。
\[4a+b\] これ以上まとめることができないので、答えは
\[4a+b(個)\]となります。 また、このゲームでよく出てくる二次方程式も、文字式で表しましょう。
\[x\times x-3\times x+10\] まずは同じ積を指数にまとめましょう。 \[x^2-3\times x+10\] つぎに、掛け算の記号を消して、数を前に持ってきましょう。
\[x^2-3x+10\] これで、文字式を整理できました！掛け算の記号を何回も書くよりも、簡潔になりましたね。

まとめ

  • 文字式とは、文字と数字を組み合わせて作られるものである。
  • 文字式を書く時には、「ルール」を守る必要がある。
  • 文字式を使うことによって、分からない数を式に含めることができる。
これで終わりです。やったね！
英語/English

What is an Algebraic Expression?

An algebraic expression is a formula that combines numbers and letters.

...Even if it's said, "What are these letters?" Let's consider this example.
Did you not solve such problems when you were in elementary school?
Person A has 10 oranges. Person B has \(\square\) apples.
What is the total number of oranges and apples?
Any number can fit into this square, right?
1, 2, 10, 100, or even 0. Any number can be placed.
(※ To be precise, \(\square\in\mathbb{N}\), the number in the square must be a natural number)
In this way, "unknown numbers" are sometimes represented as symbols like squares.
Oh, by the way, the answer is \(10+\square\).

...but that's a story up to elementary school.

Algebraic Expressions from Middle School Onwards

Until now, we have used symbols like squares and triangles, but what will we use once we reach middle school?

The answer is 'Alphabet letters'.


However, there are some rules when using alphabet letters.
Let's consider an example based on this problem.
Person A has \(a\) oranges. Additionally, Person B has three times the number of apples as Person A and \(b\) oranges.
What is the total number of oranges and apples that Person A and Person B have?
First, let's think about how many apples and oranges Person B has.
  • First, Person A has \(a\) apples.
  • Person B has 'three times the number of apples that Person A has'.
  • Additionally, Person B has \(b\) oranges.
Considering these, the answer is
\[a+3\times a + b\] However, this is not the correct answer yet..

Rules of Algebraic Expressions

So, what are these 'rules'? Here they are.
  • Omit the multiplication symbol '\(\times\)'
    If the multiplication symbol remains, it is considered as being in the middle of calculation, which can lead to deductions.
  • Numbers come before letters in terms.
    For example, \(a\times 4\times b=4ab\). Arrange the numbers at the beginning of each term.
    Do not place numbers in between or in the middle of letters like \(a4b\).
  • Arrange the product of letters in alphabetical order.
    This is not very important, but
    \(x\times a\times z\times b\) should be arranged as \(abxz\).
    Writing it as \(xazb\) does not have much effect, though.
  • Express the product of the same letters using exponents. For example, \(a\times a\times b\times c\times c\times c=a^2bc^3\).
    Instead of \(a\times a=aa\), write the number of times the same letter appears as an exponent on the right top of the letter.
    However, do not use exponents for numbers unless they are significantly large.
  • Use fractions for division. For example, \(2\times a\div b \times c=\dfrac{2ac}{b}\). Avoid using the division symbol and use fractions instead.
  • Combine like terms.
    Combine terms that contain the same letters in the same amount (e.g., \(2ab+3ab\), \(3a^3b^2c-4a^3b^2c\)).
    For example, \(2ab+3ab=5ab\), \(-a^3b^2c\).
  • Avoid using mixed fractions from elementary school; stick to improper fractions.
  • (While this game does not cover it, place constants like the number \(\pi\) after numbers but before variables in expressions.)
Following these rules, let's correct the expression \[a+3\times a + b\] First, remove the multiplication symbol, so it becomes:
\[a+3a+b\]
Combine like terms, and it becomes:
\[4a+b\] This cannot be simplified further, so the answer is:
\[4a+b\] (units). Also, let's represent a common quadratic equation in algebraic form:
\[x\times x-3\times x+10\] First, combine the same products into exponents. \[x^2-3\times x+10\] Next, remove the multiplication symbol and bring the numbers forward:
\[x^2-3x+10\] Now, the algebraic expression is organized! It's more concise than writing the multiplication symbol multiple times.

Summary

  • Algebraic expressions combine letters and numbers.
  • There are specific 'rules' to follow when writing algebraic expressions.
  • Using algebraic expressions allows us to include unknown numbers in equations.
